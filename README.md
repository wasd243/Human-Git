## English Introduction

Human-Git is an automated Git tracking and synchronization tool designed to streamline your development workflow. It silently monitors your coding activities, detects file mutations in real-time, and creates "shadow checkpoints" in a separate shadow branch without interfering with your primary Git workflow. It provides real-time visualizations of your insertions and deletions directly via an elegant desktop UI. 

Currently, Human-Git is actively under development to provide robust background processing, intelligent debouncing, and seamless Tauri-powered UI integration.

## 中文介绍

Human-Git 是一款自动化的 Git 跟踪和同步工具，旨在简化您的开发工作流。它在后台默默监控您的编码活动，实时检测文件变动，并在独立的影子分支中创建“影子存档点（shadow checkpoints）”，而绝不干扰您的主要 Git 工作流。它通过优雅的桌面用户界面，直观、实时地展示您的代码增删情况。

目前，Human-Git 正处于活跃开发阶段，致力于提供稳健的后台处理、去抖动机制以及与 Tauri UI 的无缝集成。
